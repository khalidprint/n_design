# N-DESIGN Архитектура приложения

## Структура проекта

### 📁 `/screens`
Все экраны приложения с полным функционалом и стилями.

### 📁 `/constants`
- `colors.js` - палитра цветов
- `fonts.js` - шрифты и размеры
- `icons.js` - список иконок
- `images.js` - URL изображений

### 📁 `/data`
- `projects.js` - данные проектов
- `services.js` - данные услуг
- `categories.js` - категории услуг

### 📁 `/utils`
Вспомогательные функции для форматирования и обработки данных.

### 📁 `/config`
Конфигурация навигации и тем.

## Навигационная структура

```
App
├── Bottom Tab Navigator
│   ├── Home Stack
│   │   ├── HomeScreen
│   │   └── ProjectDetailScreen
│   ├── Projects Stack
│   │   ├── ProjectsScreen
│   │   └── ProjectDetailScreen
│   ├── Services
│   │   └── ServicesScreen
│   └── Profile
│       └── ProfileScreen
```

## Компоненты и их ответственность

### HomeScreen
- Отображает основную информацию о компании
- Показывает карточки услуг
- Горизонтальный скролл последних проектов
- Секция преимуществ

### ProjectsScreen
- Список всех проектов
- Фильтрация по категориям
- Переход к деталям проекта

### ProjectDetailScreen
- Полная информация о проекте
- Характеристики (площадь, стоимость)
- Описание работ
- CTA для консультации

### ServicesScreen
- Карточки всех услуг
- Описание и особенности каждой услуги
- CTA блок

### ProfileScreen
- Информация о дизайнере
- Статистика проектов
- Меню настроек

## Стилизация

Приложение использует:
- **StyleSheet** из React Native для стилов
- **Единую палитру цветов** из constants
- **Консистентные размеры шрифтов**
- **Тень и радиусы** для глубины

## Навигация

- **Bottom Tab Navigator** для основной навигации
- **Stack Navigator** для навигации внутри разделов
- **Параметры передаются через `route.params`**

## Управление состоянием

Приложение использует:
- **React Hooks** (useState, useEffect)
- **Navigation Props** для навигации
- **Локальные данные** (constants/data)

## Лучшие практики

2. ✅ Отделяйте стили в отдельный объект

5. ✅ Тестируйте на разных размерах экранов
